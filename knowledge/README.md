# Stell Knowledge

This tree is the GitHub-managed knowledge source for Stell.

Rules:
- Keep text-first content here.
- Store approved prompts, SOPs, policies, FAQs, and product notes here.
- Do not store large binary files here.
- Heavy files belong in Google Drive and are indexed after sync.

Folders:
- `faq/`: short question and answer content
- `products/`: product, module, and feature notes
- `operations/`: admin, queue, audit, and support content
- `automation/`: delegation, WhatsApp, connector, and agent behavior notes
- `customers/`: reusable account-safe templates only
